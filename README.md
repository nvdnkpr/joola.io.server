# joola.io.server
The open-source data visualization framework

[![Build Status][3]][4] [![dependency status][5]][6] [![dev dependency status][7]][8] [![Coverage Status][1]][2]

Installing
----------

```
npm install -g joola.io.server
npm start

navigate to http://localhost:8000
```


Testing
-------

```
npm test
```
